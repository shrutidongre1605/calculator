# Simple Calculator

A simple calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is designed to demonstrate fundamental programming concepts, including user input handling, mathematical calculations, and error handling.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⚠️ Division by zero handling
- 🎯 Simple and user-friendly interface
- 💻 Lightweight and easy to run

```bash

```

3. Open `index.html` in your browser.

## 🎮 Usage

1. Enter numbers using the calculator buttons.
2. Select an arithmetic operation.
3. Press the equals (`=`) button to view the result.
4. Use the clear (`C`) button to reset the calculator.

# Calculator

A simple calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is designed to demonstrate fundamental programming concepts, including user input handling, mathematical calculations, and DOM manipulation.

## 🌐 Live Demo

https://shrutidongre1605.github.io/calculator/

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🗑️ Delete Last Character
* 🧹 Clear Screen
* ⚠️ Error Handling
* 🎯 Simple and User-Friendly Interface
* 💻 Lightweight and Easy to Run

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/shrutidongre1605/calculator.git
```

2. Navigate to the project folder:

```bash
cd calculator
```

3. Open `index.html` in your browser.

## 🎮 Usage

1. Enter numbers using the calculator buttons.
2. Select an arithmetic operation.
3. Press the equals (`=`) button to calculate the result.
4. Use the delete button (`⌫`) to remove the last character.
5. Use the clear button (`C`) to reset the calculator.
